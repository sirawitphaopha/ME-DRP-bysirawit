### 3. ลบไม่ได้จากเว็บ → "ซ่อน" แทน
- migration `0010`: ถอน policy delete ของ anon (ลบจริงเฉพาะใน Supabase = พี่กันคนเดียว)
- migration `0011_drugs_hidden`: คอลัมน์ `hidden boolean default false` (**พี่กันรันเองผ่าน SQL Editor** — MCP ติดอนุมัติหลัง reconnect)
- `setDrugHidden(d, hidden)` ผ่าน `updateDrug` · หน้าคลังยา + autocomplete กรอง `!d.hidden` ออก ·
  หน้าตั้งค่ามีส่วน "💊 ยาที่ซ่อนอยู่" (เอากลับมาแสดง/ประวัติ) · footer คลังยาบอก "ซ่อนอยู่ N รายการ"

### 4. Audit log (ประวัติการแก้ไข)
- migration `0010_drug_audit`: ตาราง `drug_audit` + trigger `log_drug_change` (SECURITY DEFINER)
  จับทุก insert/update/delete → เก็บ old/new (jsonb) + เวลา · **จับได้แม้แก้ตรงใน Supabase/MCP**
- RLS: อ่านได้ (anon) · เขียนได้เฉพาะ trigger
- `fetchDrugAudit` + `renderDrugLogModal` — ไทม์ไลน์ + diff ราย field ("ชื่อยา: เก่า→ใหม่", "การแสดงผล: แสดง→ซ่อน")
- เหตุผล: พี่กันอยากรู้ว่ามีใครแอบเปลี่ยนชื่อไหม (แอปไม่มี login เลยระบุ "ใคร" ไม่ได้ · แต่เก็บ "อะไร+เมื่อไหร่" ครบ)
