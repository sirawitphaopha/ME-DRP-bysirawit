5. **Responsive มือถือ** (เดสก์ท็อปไม่แตะ — แยก branch ด้วย `isMobile` = matchMedia max-width:640px)
   - หัวเมนู: มือถือ 2 แถว (โลโก้+ชื่อสั้น+v0.9.1 + เฟือง⚙=ตั้งค่า / ปุ่ม กรอก·รายการ·สรุป เต็มแถว)
   - หน้ารายการ: การ์ดแนวตั้ง แทนตาราง min-width:860px
   - Dashboard: KPI 4→2 คอลัมน์ · กราฟ/breakdown ยุบแนวตั้ง · ตารางล่าสุด→การ์ด
6. **เวลาที่พบ → เวร (auto-detect) + นาฬิกาเดินจริง**
   - เอาช่องนาฬิกา HH:MM ออก → ปุ่มเลือกเวร 3 ปุ่ม (เวรเช้า/เวรบ่าย/เวรดึก)
   - auto ตามเวลาปัจจุบัน: 08–16 เช้า · 16–24 บ่าย · 00–08 ดึก (ใช้ `shiftOf` เดิม) · กดสลับเองได้ (set `occurred_time` เป็นเวลาตัวแทน → ตรรกะบันทึก `shift` เดิมไม่ต้องแก้)
   - เพิ่ม `clock` state + setInterval 1s (เฉพาะหน้ากรอก) แสดง `⏱ HH:MM:SS`
   - หน้ารายละเอียดแสดงแค่ "เวร" ไม่โผล่เวลาตัวแทน
7. **bump 0.9 → 0.9.1** (แสดงหัวเว็บทั้งเดสก์ท็อป + มือถือ)
