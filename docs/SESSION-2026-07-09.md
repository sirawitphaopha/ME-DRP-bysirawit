## สรุปสิ่งที่ทำ

1. **โคลน repo ลงเครื่อง** (`git clone`) + `npm install` (Next.js 15.5.20, node v24)
2. **แก้ปัญหา deploy Cloudflare ไม่ขึ้น** — ต้นเหตุ: Build command บน Cloudflare เป็น `npm run build` (ได้แค่ `.next/` ไม่ได้ `.open-next/`) → OpenNext deploy หาไฟล์ compile ไม่เจอ (`Could not find compiled Open Next config`)
   - **วิธีแก้ (ตั้งบน Cloudflare Dashboard → Settings → Build):** Build command = `npx opennextjs-cloudflare build` · Deploy command = `npx wrangler deploy` · deploy สำเร็จ
   - ตั้ง env `NEXT_PUBLIC_SUPABASE_URL` + `NEXT_PUBLIC_SUPABASE_ANON_KEY` (Variables and secrets)
3. **`.env.local`** — สร้างให้เว็บในเครื่อง/มือถือต่อ Supabase จริง (ไม่ใช่ demo) · gitignored ไม่ขึ้น git
4. **ผู้รายงาน → เมนูเลือก 16 คน** — เพิ่ม `REPORTERS` ใน `lib/constants.ts` (เจ้าหน้าที่ห้องยา OPD 16 คน · ตำแหน่งย่อขึ้นหน้า ภก./ภญ./จพ./พนง.) · ช่องผู้รายงาน (หน้ากรอก + โหมดแก้ไข) เปลี่ยนจากพิมพ์เอง → `HSelect`
5. **Responsive มือถือ** (เดสก์ท็อปไม่แตะ — แยก branch ด้วย `isMobile` = matchMedia max-width:640px)
   - หัวเมนู: มือถือ 2 แถว (โลโก้+ชื่อสั้น+v0.9.1 + เฟือง⚙=ตั้งค่า / ปุ่ม กรอก·รายการ·สรุป เต็มแถว)
   - หน้ารายการ: การ์ดแนวตั้ง แทนตาราง min-width:860px
   - Dashboard: KPI 4→2 คอลัมน์ · กราฟ/breakdown ยุบแนวตั้ง · ตารางล่าสุด→การ์ด
6. **เวลาที่พบ → เวร (auto-detect) + นาฬิกาเดินจริง**
   - เอาช่องนาฬิกา HH:MM ออก → ปุ่มเลือกเวร 3 ปุ่ม (เวรเช้า/เวรบ่าย/เวรดึก)
   - auto ตามเวลาปัจจุบัน: 08–16 เช้า · 16–24 บ่าย · 00–08 ดึก (ใช้ `shiftOf` เดิม) · กดสลับเองได้ (set `occurred_time` เป็นเวลาตัวแทน → ตรรกะบันทึก `shift` เดิมไม่ต้องแก้)
   - เพิ่ม `clock` state + setInterval 1s (เฉพาะหน้ากรอก) แสดง `⏱ HH:MM:SS`
   - หน้ารายละเอียดแสดงแค่ "เวร" ไม่โผล่เวลาตัวแทน
7. **bump 0.9 → 0.9.1** (แสดงหัวเว็บทั้งเดสก์ท็อป + มือถือ)
