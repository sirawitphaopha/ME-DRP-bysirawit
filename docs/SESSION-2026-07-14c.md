   (`efArr`/`efToggleArr` · toggle chip เหมือนหน้ากรอก)
3. **ลบเคสที่ยังไม่ขึ้นระบบแล้วเด้งกลับ** — `doSoftDelete` เรียก `dequeuePending(rec.id)` ด้วย
4. **แก้ไขตอนเน็ตหลุด ขึ้น "บันทึกแล้ว" หลอก** — `saveEdit` ใช้ `pushUpdate` (ยืนยันผล + ลองซ้ำ
   3 รอบ · AbortController) เดิมดัก error เงียบ
5. **หน้าแก้ไขไม่ตรวจช่องบังคับ** — `saveEdit` เรียก `validateIncident` เหมือนหน้ากรอก
6. **คิว/แคช race** — `mutatePending` (อ่าน-แก้-เขียนทีละรายการ ไม่ทับทั้งชุด) + `dedupUnsynced`
   (คงเคสที่ยังไม่ซิงก์: pending + id ที่ยังไม่ใช่ uuid)
22. `flushPending` พลิกหน้าผล `fail→ok` ถ้าคิวส่งเคสของหน้าผลสำเร็จ

24. `saveEdit` คง `shift` เดิมถ้าไม่มีเวลาใหม่ (`shiftOf(occurred_time) || ef.shift || det.shift`)

9. **ตารางรายงานล่าสุด Dashboard** — ใช้ `natureText(r.error_type)` (เดิมโชว์ array ดิบ ติดกัน)

## ชุด D 🟡 — เก็บกวาดขอบเคส (commit `b2573f1`)

12. **AN ค้างเมื่อออกจาก IPD** — `setLocation`(ฟอร์ม)/`setEfLocation`(แก้ไข) ล้าง `an` เมื่อ ≠ IPD
13. **ธง High-alert auto ค้าง** — state `hadAuto` แยก auto จากยา HAD vs ผู้ใช้ติ๊กเอง ·
    `hasHadDrug`(เช็ค "(HAD)" ใน drug line)/`clearAutoHad`(ปลดธงเมื่อลบยา HAD หมด · ถ้า auto) ·
    `toggleHighAlert` ติ๊กเอง = ปลด `hadAuto` (ไม่ลบธงที่ตั้งใจ) · reset ตอน save/สลับประเภท
14. **"อื่น ๆ" ไม่บังคับระบุ + ข้อความค้าง** — `validateIncident` เพิ่ม `error_nature_other`/
    `drp_type_other` · `toggleNature`/`setDrpType` ล้างข้อความเมื่อเลิกเลือก + โชว์ error ใต้ช่อง
15. **AN auto-format** (โจทย์พี่กัน) — `formatAn(raw, occurredAt)` → "YY-NNNNN" ตอน blur ·
    YY = ปี พ.ศ. 2 หลักจาก **ปีที่เกิดเหตุ** (occurred_at + 543) · NNNNN = 5 หลัก zero-pad ·
    พิมพ์ "1234"→"69-01234", "1"→"69-00001" · พิมพ์เกิน 5 หลัก = 5 ตัวท้ายเป็นลำดับ ที่เหลือเป็นปี ·
    `save` normalize AN เสมอ (เผื่อไม่ blur) · validation กัน "-" เปล่า (`/\d/`)
16. **ร่างอัตโนมัติหาย** — `hasDraftContent` รวม severity/intervention/outcome/an
17. **ค้นหาเจอมั่ว** — `matchSearch` (whitelist: hn/an/reporter/location/ยา/detail/cause/
    management/natureText/drpLabel/drp_type_other) แทน `JSON.stringify` ทั้งก้อน (ทั้ง Dashboard + รายงาน)
18. **CSV formula injection** — `esc` เติม `'` นำหน้าช่องที่ขึ้นต้น `= + - @`
19. **ถังขยะไม่สดข้ามเครื่อง** — `refreshTrashIfOpen`(loadTrash เมื่อ view=manage) ใน realtime/
    visibility/online · `restoreIncident` คืน boolean (`.select("id")`) → กู้เคสที่โดนลบถาวรแล้วบอกตามจริง
20. **ปุ่มถังขยะไม่โชว์สถานะ** — กู้คืน/ลบถาวร disabled ตอน `trashBusy` + ปุ่มลบเป็น "กำลังลบ…"
21. **ป๊อปลบถาวรไม่บอกเคส** — โชว์ประเภท/HN/วันที่/ผู้รายงาน/ยา (กัน HN ซ้ำลบผิด)
25. เก็บกวาด: กัน "ระดับ —" ในการ์ดรายงานล่าสุด (เคสไม่มี severity) · `audio resume().catch()`
    กัน unhandled rejection · (heatmap TZ เพี้ยนเฉพาะโซนเวลาติดลบ — ไม่กระทบไทย ไม่แก้)
