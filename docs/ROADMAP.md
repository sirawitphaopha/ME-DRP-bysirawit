## 🔜 งานถัดไป (ตกลงแล้ว) — ปรับหน้าตั้งค่า

**เหตุผล:** ผู้ใช้จริง = เภสัชกร ไม่ควรเห็นช่องกรอก Supabase URL/key (งง + เสี่ยงพลาดจนต่อไม่ติด + ตอนนี้ตั้งผ่าน env แล้ว) → ซ่อน config ฐานข้อมูล แล้วเอาของที่ใช้จริงมาใส่

### กลุ่ม A — ทำก่อน (เร็ว ไม่ต้องแตะฐานข้อมูล)
- [ ] **เกี่ยวกับแอป** — ชื่อ · เวอร์ชัน (`APP_VERSION`) · ห้องยา OPD รพ.ปรางค์กู่ · คำอธิบายสั้น
- [ ] **สถานะระบบ** (อ่านอย่างเดียว) — ● เชื่อมต่อฐานข้อมูล / ● โหมด demo · จำนวนเคสในระบบ (Med / DRP)
- [ ] **เครื่องมือ** — ปุ่มล้างข้อมูล demo ในเครื่อง · Export CSV ทั้งหมด
- [ ] **การเชื่อมต่อ (ขั้นสูง)** — ย้าย URL/key มา**พับซ่อน** · โชว์เฉพาะตอน**ยังไม่เชื่อมต่อ** (ตั้งครั้งแรก) หรือกดปุ่ม "ขั้นสูง"

### กลุ่ม B — เฟสถัดไป (ต้องทำตาราง Supabase + API เพิ่ม)
- [ ] **จัดการรายชื่อผู้รายงาน** — เพิ่ม/ลบ/แก้ 16 คน เองได้ (ตอนนี้ hardcode ใน `lib/constants.ts` `REPORTERS`)
  - 🔑 ถ้าอยากให้**ทุกเครื่องเห็นรายชื่อเดียวกัน** → เก็บใน Supabase (ตารางใหม่ เช่น `reporters`) · ถ้าเก็บ localStorage จะไม่ sync ข้ามเครื่อง
- [ ] **จัดการค่า dropdown** — ประเภท Error / DRP / Intervention / จุดที่พบ แก้เองได้ (เก็บ Supabase เช่นกัน)
