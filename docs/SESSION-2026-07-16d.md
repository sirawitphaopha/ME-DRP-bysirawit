## วิธีแก้
- **`incidents.drug_ids jsonb`** (migration `0012` · **applied** — พี่กันรัน SQL Editor) เก็บ `drugs.id` ขนานกับ `drugs[]`
- `pickDrug` เก็บ id · พิมพ์เอง=null · `save`/`saveEdit` จับคู่ข้อความ+id ตาม index (คงการผูก)
- **`resolveDrugLines(r, byId)`** (lib/helpers) — แปลงรายการยาเป็น "ชื่อล่าสุด": มี id + เจอในคลัง → `drugFlatLine` ปัจจุบัน ·
  ไม่มี/เคสเก่า/พิมพ์เอง → ข้อความเดิม · ใช้ทุกจุด (หน้ารายงาน/รายละเอียด/Dashboard recent/CSV)
- **Dashboard `byDrug` group ด้วย id** (fallback ข้อความ · label = ชื่อล่าสุด) → เปลี่ยนชื่อไม่นับซ้ำ
- **`backfillDrugIds`** — จับคู่รหัสให้เคสเก่าอัตโนมัติ (ครั้งเดียว/เครื่อง · guard localStorage · exact match
  `drugFlatLine` · เคสที่ยาเปลี่ยนชื่อไปแล้ว/พิมพ์เอง แมตช์ไม่ได้ → ปรับเองใน Phase 3)
